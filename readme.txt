wp show category posts show's posts from any category


== Description ==

wp show category shows any category posts in widget , page or post, have shortcode [wpscp]



== Installation ==

The plugin can be use just by installing it by wordpress admin or upload manually. 

1. Upload `wp-show-category-posts` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now go to wp-show-category-posts tab at wp-admin

== Frequently Asked Questions ==

= How to set number of posts ? =

 Go to "wp show category posts" in left panel of wordpress. enable or disable title, thumbnail, or date as you need or other setting.
 and press "Save Changes" button
 
 = How to use plugin on page or posts ? =

 use [wpscp] shortcode in page, post or widget.

For more visit blog [wp-show-category-posts](https://ajaysharma3085006.wordpress.com/ "wp-show-category-posts")
